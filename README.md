This is a simple Contact Management System developed using C++. The project allows users to manage contacts efficiently through basic CRUD operations including adding, viewing, searching, updating, and deleting contacts. It also uses file handling to store data permanently.

The project was built to practice C++ fundamentals, object-oriented programming concepts, and file handling techniques. It helps in understanding how to structure a medium-sized console application and manage data in a simple system.

Technologies used: C++, File Handling, OOP basics.

Main files in the project:
- main.cpp (main logic)
- contact.cpp (contact operations)
- header.h (function declarations)
- Visual Studio project files

To run the project, open it in Visual Studio, build the solution, and run the program.
